FitPet — sprites del avatar interactivo
========================================

Coloca aquí tus 3 imágenes del personaje (PNG con fondo transparente
idealmente). El componente InteractivePet las carga por estos nombres:

  stand.png  ->  de pie con gafas   (idle / caminar / correr)
  flex.png   ->  flexionando        (saltar / celebrar / reaccionar / arrastrar)
  sit.png    ->  sentado            (dormir / descansar)

Mientras falte algún PNG, el avatar usa un emoji (🐸) como respaldo,
así que la app sigue funcionando sin las imágenes.

Recomendado: imágenes cuadradas ~256x256 px, fondo transparente.
